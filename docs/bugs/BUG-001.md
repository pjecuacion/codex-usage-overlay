# BUG-001 - Renderer Could Not Read Codex Usage

## Context
The Windows overlay launched, but the renderer showed blank usage values and the error `Cannot read properties of undefined (reading 'getSnapshot')`.

## Symptoms
- Remaining usage displayed as `--%`.
- Sessions and messages displayed as `-- / --`.
- Status showed `Could not read local Codex usage`.

## Root Cause
The Electron app uses `"type": "module"` for the main process, but Electron loaded the preload file as a CommonJS preload script. Because `src/main/preload.js` used ESM `import`, preload execution failed before exposing `window.codexUsage`.

## Fix
- Replaced `src/main/preload.js` with `src/main/preload.cjs`.
- Updated the main process and smoke test to point at `preload.cjs`.
- Added renderer-level smoke verification.
